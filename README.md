# AflaChat - Your AI Partner Against Aflatoxin 🌽🛡️

AflaChat is an AI-powered conversational platform dedicated to ensuring food safety by providing instant, reliable information and guidance on Aflatoxin prevention and management.

---

## ✨ Features

- 💬 **AI-Powered Chat**: Instant access to Aflatoxin safety protocols and expert advice.
- 🌍 **Multilingual Support**: Fully localized in **English** and **Swahili** to reach more farmers and consumers.
- 📚 **Comprehensive Guidance**: Practical tips for pre-harvest, harvest, and post-harvest management.
- 🛡️ **Safety Awareness**: Educational content for identifying contamination and ensuring food quality.
- 📱 **Responsive Design**: Modern, mobile-first interface built for accessibility in the field.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language Management**: Custom Context-based localization system.

---

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

## 📁 Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable React components (Language selector, buttons, etc.).
- `lib/`: Utility functions, translations, and global state logic.
- `public/`: Static assets including logos and icons.
